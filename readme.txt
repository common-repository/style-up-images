Style Up Images helps you to make your images looks little more good & informative on mouseover.

== Description ==

Style Up Images add some effect to your images on mouseover. It will shown the description of image from alt text.

= How to use =
1. Check the preview & usage instructions [here](http://www.eworksphere.com/style-up-images/). Take your mouse over the image.

== Installation ==

= Installation =

1. You can use the built-in installer.
     OR         
     Download the zip file and extract the contents.
     Upload the 'style-up-images' folder to your plugins directory (wp-content/plugins/).
1. Activate the plugin through the 'Plugins' menu in WordPress.
